# E-commerce-Customer-Behavior
Mengimplementasikan Machine Learning untuk prediksi harga rumah menggunakan algoritma Decision Tree dan Linear Regression dengan optimasi parameter menggunakan Genetic Algorithm.

## Dataset
- Jenis data: Data tabular harga rumah  
- Target: Harga rumah  
- Fitur: Atribut numerik yang merepresentasikan karakteristik rumah  

---

## Metodologi
1. Exploratory Data Analysis (EDA)  
2. Data Preprocessing  
3. Implementasi Decision Tree dan Linear Regression  
4. Optimasi Hyperparameter dengan Genetic Algorithm  
5. Evaluasi Model  
6. Analisis Hasil dan Kesimpulan  

---

## Model & Evaluasi
**Model yang digunakan:**
- Decision Tree  
- Linear Regression  

**Optimasi:**
- Genetic Algorithm (GA)
